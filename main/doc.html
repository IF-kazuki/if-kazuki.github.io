.Introduction

please put the above URL inside the head tag.

next, create a div tag in index.html that will serve as a reference for creating the DOM, and add root to the id tag.

prepare a script tag under the div tag and call a function named SPA in it. the argument represents the directory structrure in JSON format 

      SPA({
        header: {
          "/": "header.html",
        },
        main: {
          "/": "index.html",
          "/doc": "doc.html",
        },
        footer: {
          "/": "footer.html",
        },
      });

the header main footer represents the folder name, and the "/" and "/doc" in it represent the URL. then describe the corresponding file name.

also, by calling the SPA function, an object called SPA is added to the window object. it contains various functions used in SPA. it is also possible to get the return value of the SPA function. 

      const { Link, Import, Cacher, DevMode } = window.SPA;

the link function selects the id tag to be given to the firest argument, the URL of the transition destination in the second argument, and the tag to be updated in the thired argument. 

      Link("main-doc", "/doc", "main");

the import function, describe the URL to be acquired in the first argument and the tag name to be given in the second argument. 

      Import("components/codeComponent.html", "codeComponent");

the cacher function can select the HTML file to be acquired in advance during the production. 

      Cacher();

the devmode function desplays the page you want to desplay as an index page. used during development. 

      DevMode("/doc");
